## 图片样式优化建议

### 当前样式问题
1. **缺少固定宽高比**：可能导致图片变形
2. **尺寸可以优化**：240px 可能偏小

### 建议优化方案

#### 方案1：保持当前尺寸，添加固定宽高比（推荐）
```css
body.activities .event-thumb {
    width: 100%;
    max-width: 240px;
    aspect-ratio: 4 / 3; /* 固定宽高比 */
    margin-bottom: 12px;
    overflow: hidden;
}

body.activities .event-thumb img {
    width: 100%;
    height: 100%;
    object-fit: cover; /* 保持比例，裁剪填充 */
    display: block;
    opacity: 0.85;
    filter: grayscale(20%);
}
```

#### 方案2：稍微增大尺寸，保持克制
```css
body.activities .event-thumb {
    width: 100%;
    max-width: 280px; /* 从 240px 增加到 280px */
    aspect-ratio: 4 / 3;
    margin-bottom: 12px;
    overflow: hidden;
}
```

#### 方案3：保持现状
- 当前实现已经很好
- 降权处理合适
- 不需要改动
