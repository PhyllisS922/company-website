# 全站翻译后端API

这是一个简单的后端服务，用于代理OpenAI翻译API调用。

## 架构

```
前端 (浏览器) 
  ↓ HTTP请求
后端API (Node.js/Python)
  ↓ 使用API密钥
OpenAI API
  ↓ 翻译结果
后端API
  ↓ 返回结果
前端 (浏览器)
```

## 部署选项

### 选项1: Vercel (推荐，最简单)
- 免费额度充足
- 自动部署
- 无需服务器

### 选项2: Netlify Functions
- 免费额度充足
- 自动部署
- 无需服务器

### 选项3: Railway/Render
- 简单易用
- 有免费额度
- 需要注册账号

### 选项4: 自己的服务器
- 完全控制
- 需要维护

## 实现步骤

1. 选择部署平台（推荐Vercel）
2. 创建API密钥环境变量
3. 部署代码
4. 更新前端代码使用新的API端点

## 成本

- 后端服务：免费（使用免费平台）
- OpenAI API：约 $0.01-0.02/次翻译（取决于内容长度）
- 总成本：非常低
